Detailed Description

CAN-интерфейс для отправки управляющих сообщений

  • Поддержка стандартных CAN ID (11-бит)
  • Формирование сообщений с проверкой целостности
  • Настройка DLC и типа фрейма

Warning
ID должен быть валидным (не превышать 0x7FF)
Note
Сообщение всегда имеет DLC=8 (8 байт данных)

Создает CAN-сообщение для управления устройством

  • Данные размещаются в байтах 1 и 3
  • Если PID < 0: speed в байте 1, checksum в байте 3
  • Если PID >= 0: checksum в байте 1, speed в байте 3
