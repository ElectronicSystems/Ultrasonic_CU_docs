Detailed Description

Bootloader для AT32F413RCT7 с CAN-интерфейсом

  • Реализация двухэтапной загрузки прошивки
  • Поддержка CAN 2.0B с аппаратным ускорением
  • Механизм защиты от сбоев при обновлении

◆ USBFS_L_CAN1_RX0_IRQHandler()

void USBFS_L_CAN1_RX0_IRQHandler ( void )

Обработчик прерывания приёма сообщений по CAN1 RX0.

Note
Реализует основную логику работы бутлоадера:
  • Прием и обработка управляющих команд (CMD_HOST_INIT, CMD_PAGE_PROG, CMD_BOOT)
  • Прием и запись данных страниц во флеш-память
  • Проверка CRC каждой страницы и всей программы
  • Переход в основное приложение при успешной валидации

Поведение по состояниям:

  • PAGE_PROG:
    • Прием данных страниц в буфер PageBuffer
    • Проверка CRC после накопления полной страницы
    • Программирование страницы во флеш или отправка ошибки
  • IDLE:
    • Прием команд:
      • CMD_HOST_INIT — установка флага обновления и готовность к прошивке
      • CMD_PAGE_PROG — подготовка буфера для приема новой страницы
      • CMD_BOOT — проверка CRC всей программы и подготовка к запуску приложения

Структура пакета:

  • Байт 0: Команда или данные
  • Байт 1: Индекс страницы (при CMD_PAGE_PROG)
  • Байт 2-5: CRC страницы или программы
