void CanOutput_Task ( void * pvParameters)

Задача отправки CAN-сообщений

Parameters
pvParametersПараметры задачи (не используются)

Выполняет:

  • Создаем speed и checksum
  1. Если CAN_BLOCK_BIT=0, то берем я pid из очереди xQueuePeek, иначе не перезаписываем speed и checksum, pid = 0.

  • Отправляем сообщение в CAN шину

◆ SensorWorking_Task()

void SensorWorking_Task ( void * pvParameters)

Задача обработки данных датчиков

Parameters
pvParametersПараметры задачи (не используются)

ЗАДАЧА ЗАПУСКАЕТСЯ ТОЛЬКО ПОСЛЕ ТОГО КАК ПРИЛИ ДАННЫЕ С ОБОИХ ДАТЧИКОВ и выполняет::

  • Триггерный сигнал на датчики.
  • По флагу dma_rx_done определяет что данные пришли из DMA.
  • Вычисляет значение distances[i] в цикле.
  • Управляет флагом CAN_BLOCK_BIT, он необходим чтобы слать в посылку нули при условии что хотя бы один датчик не дает данные.
  • Если все датчики ответили то среднее значение с двух датчиков уходит в очередь ProcessedDataQueue
