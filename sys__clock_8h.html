void system_clock_config ( void )

system clock config program

Note
the system clock is configured as follow: system clock (sclk) = hext / 2 * pll_mult system clock source = pll (hext)
  • hext = HEXT_VALUE
  • sclk = 192000000
  • ahbdiv = 1
  • ahbclk = 192000000
  • apb2div = 2
  • apb2clk = 96000000
  • apb1div = 2
  • apb1clk = 96000000
  • pll_mult = 48
  • pll_range = GT72MHZ (greater than 72 mhz)
